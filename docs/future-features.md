# Future Features Roadmap

This document records ideas that are planned for future versions of Trading Workspace.

---

# Workspace

## Draggable Panels

Allow users to freely arrange:

* Chart
* Replay
* Order Entry
* Position Card
* Heatmap
* CVD
* Funding
* Watchlist

Support multiple saved workspace layouts.

---

# Keyboard Engine

Every feature should support keyboard operation.

Examples:

* Long
* Short
* Close Position
* Replay
* Leverage
* Margin
* TP / SL

---

# Custom Key Binding

Users should be able to customize every shortcut.

Examples:

A → Long

S → Short

Space → Replay

Users may change them to any preferred keys.

Support:

* Save
* Import
* Export
* Reset

---

# Floating Command Menu

Inspired by:

* Visual Studio Code Command Palette
* Bloomberg Terminal
* Professional Game UI

Example:

Press W

↓

Position Size

* Amount
* Margin
* Notional

After selection, the floating window closes automatically.

---

# Replay Simulation

Support replay from any historical period.

Features:

* Play
* Pause
* Previous
* Next
* Playback Speed

---

# What-if Analysis

Modify future candles during replay.

Examples:

* Increase next 3 candles by 5%
* Decrease next 10 candles by 8%

Automatically recalculate:

* EMA
* MA
* RSI
* MACD
* Liquidation Price

---

# Exchange Integration

Support multiple exchanges.

Planned:

* Binance Futures
* Bybit
* OKX

Trading modes:

* Paper Trading
* Live Trading

---

# Plugin System

Future versions may support third-party plugins.

Examples:

* Custom Indicators
* Strategy Modules
* Alert Modules
* AI Analysis

---

# Long-Term Vision

Trading Workspace aims to become a modular trading operating system rather than a traditional charting platform.

The platform should provide maximum flexibility for discretionary traders, quantitative researchers, and strategy developers.
